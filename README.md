# 📈 IPO Market Listing Gain Prediction

A Deep Learning-based predictive system designed to forecast listing gains for Indian IPOs. This project leverages **PyTorch** for model architecture and **Streamlit** for a real-time interactive dashboard.

## 🚀 Key Features
* **Regularized Neural Network:** Uses a deep learning model with Dropout and L2 regularization to prevent overfitting.
* **Ablation Study:** Detailed comparison against Linear Regression baselines.
* **Interactive UI:** Real-time predictions via a Streamlit web application.
* **Data Pipeline:** End-to-end preprocessing of financial datasets including imputation and scaling.

## 🛠️ Tech Stack
* **Framework:** PyTorch (Neural Networks)
* **Deployment:** Streamlit
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib
* **Environment:** AntiGravity / Google Colab

## 📊 Performance
Achieved an $R^2$ Score of 0.6961 and a Mean Absolute Error (MAE) of 11.41, demonstrating strong predictive reliability in forecasting Indian IPO listing gains compared to baseline models.
